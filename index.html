Event Pages

New events will be published at /events/<slug>/index.html.

How this works

  1. Your Custom GPT collects event details.
  2. It posts JSON + HTML to a Make webhook.
  3. Make commits the files to this repository under /events/<slug>/index.html.
  4. Netlify rebuilds on commit and serves the page.

Try opening a sample (once created): /events/sample-event/
